# HFUT-DA-practicum
Hands-on materials for HFUT Data Analytics practice course. HFUT 数据分析实践课课程资料：实验、项目、数据集与代码示例。


## 主要内容

- **Prompt 设计与效果对比**  
  通过多种提示词策略进行实验，比较不同方案在任务效果、稳定性和可解释性上的差异。

- **API 驱动的数据分析流程**  
  学习将大模型能力接入分析任务，完成批量处理、结果回写和分析报告生成等实践环节。

- **SFT 数据集构建与样本规范**  
  了解指令数据组织方式，掌握基础的数据构造、样本格式与质量控制要点。

- **案例化项目训练**  
  结合课程案例完成从需求理解、方案设计到结果复盘的全过程训练，提升项目实战能力。


## 仓库内容概览

- `api_examples/`：课程中的 API 应用与分析案例资料。
- `data/`：课程实验与案例所使用的数据文件。
- `sft/`：SFT 相关样本、数据转换与数据集文件。
- `docs/`：课程说明与参考文档。
- `vibe_coding_case/`：课程中的综合实践示例。
